🧠 FitAI Pro — AI Fitness Tracker

FitAI Pro is an advanced AI-powered fitness tracking web app built using Streamlit. It helps users monitor health metrics, track nutrition, generate personalized workout plans, and gain smart insights — all in one place.

🚀 Features
🏋️ Workout Tracking
Multiple workout types:
🏠 Home Workout (No Equipment)
🧘 Yoga / Stretching
💃 Dance / Zumba
🏋️ Gym Workouts
Tracks:
Duration
Calories burned (MET-based calculation)
Smart constraint: No-equipment mode strictly avoids gym exercises
🧠 AI Workout Planner
Generates personalized 7-day workout plans
Based on:
Age, Weight, Goal
Workout type
Duration & frequency
Enforces equipment rules (e.g., no dumbbells in home mode)
🥗 Nutrition Tracker
🔹 Modes Available:
🟢 Veg Database (Indian foods)
🔴 Non-Veg Database
✏️ Manual Entry
📊 Tracks:
Calories
Protein / Carbs / Fat
Veg vs Non-Veg split
Daily calorie goal progress
🔥 Streak Tracker
Tracks consecutive active days
Motivational fire streak system
📊 Analytics Dashboard (Single Graph)
All metrics in one unified dashboard:
⚖️ Weight trend
🧮 BMI
🔥 Calories (burned vs intake)
💧 Water intake
😴 Sleep hours
🏃 Workout duration
💡 Smart Insights
Detects:
Low activity
Poor sleep
Calorie imbalance
AI-generated recommendations
📥 Export Feature
Download your full fitness data as CSV
🛠️ Tech Stack
Frontend/UI: Streamlit
Data Handling: Pandas
Visualization: Plotly
API (Optional): Groq (for AI features)
Environment Management: python-dotenv
